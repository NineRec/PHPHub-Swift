# PHPHub-Swift

PHPHub 已经有了iOS 客户端，本项目是对现有版本的一个 Swift 复刻。

其主要目的是锻炼自己的 Swift 技能。

### 主要任务

- [x] Initialize the framework
- [x] WKWebView
- [ ] Apple Push Notification Service
- [ ] Scan to login

PHPHub-Swift is the iOS application for PHPHub written with Swift.
